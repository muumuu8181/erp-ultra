# 042 Sales Commission

Module for managing sales commissions.

## API Endpoints

- `POST /api/v1/sales-commission/` - Create a commission
- `GET /api/v1/sales-commission/{id}` - Get a commission by ID
- `PUT /api/v1/sales-commission/{id}` - Update a commission
- `DELETE /api/v1/sales-commission/{id}` - Delete a commission
- `GET /api/v1/sales-commission/` - List commissions
