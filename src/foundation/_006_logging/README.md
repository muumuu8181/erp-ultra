# 006_logging

Structured Logging module for Ultra ERP.

## Usage

This module provides standard structured logging endpoints and logic, storing logs in the database.

## Endpoints

- `POST /api/v1/logging/` - Create log
- `GET /api/v1/logging/` - List logs (paginated)
- `GET /api/v1/logging/{id}` - Get log by ID
- `PUT /api/v1/logging/{id}` - Update log
- `DELETE /api/v1/logging/{id}` - Delete log
